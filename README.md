# SysProgLabs

## Lab 1

### Task: Create analogue of find func in POSIX-like systems

Usage: 
```
dirwalk path -flags
```

Possible flags:
* -f - show files
* -d - show directories
* -l - show links
* -s - sort according to LC_COLLATE

Parameters l, d, and f are automatically set if none of them was provided.
